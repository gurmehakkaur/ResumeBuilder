
# Installation Guide

Follow these steps to set up LazyMe locally:

## 1. Install Dependencies

Run the following command in the `web-app` directory:

```
npm install
```

## 2. Environment Variables

Make sure you have a `.env` file in the `web-app` directory with all required secrets and environment variables. Refer to the project documentation or your team lead for the necessary values.

## 3. Start the Development Server

After installing dependencies, start the development server:

```
npm run dev
```
